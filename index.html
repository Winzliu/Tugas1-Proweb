       Halo guys, saya ingin merangkum pembelajaran Pemrograman Web minggu kedua kami. Kami diajarkan mengenai banyak hal mulai dari apa itu HTML sampai penggunaan HTML untuk membuat website. Kami diajarkan mengenai tag tag dasar yang berguna untuk membuat website,misalnya:

  • <H1> sampai <H6> untuk menulis judul.
  • <p> untuk menulis paragraf
  • <b>,<strong>,<i>,<em>,<u> untuk memberikan style pada tulisan
  • <Br> untuk memberikan baris baru pada tulisan
  • <a> untuk membuat hyperlink yang menuju ke situs lain
  • <img> untuk menambahkan gambar
  • <ul>,<ol>,<li> untuk membuat list
  • <table>,<tr>,<th>,<td>
    • untuk membuat table.
      Dan masih banyak lagi misal di dalam table kita bisa menambahkan colspan dan rowspan untuk menggabungkan antar kolom atau baris nya. Juga ada target pada tag <a> untuk menentukan pembukaan situs selanjutnya di tab yang baru atau tidak.
